# CsvParser

If this is the first time running, do the following in the terminal:

```bash
npm install
```

# Starting

** make sure you are in the csvParser folder **
To start the application, open two terminals and input the following commands into each one respectively.

Front end application
```bash
ng build --watch
```

Backend Application:
```bash
nodemon Backend/app.ts
```

Once the server is running, open your browser and navigate to `http://localhost:4200/` and you should be able to see the parser.
